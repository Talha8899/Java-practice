# ☕ Java Practice Programs

A growing collection of Java programs written while learning Object-Oriented Programming and core Java concepts during my Software Engineering degree.

Each file focuses on a specific concept — committed regularly as new topics are studied.

---

## 📚 What's Covered

### 🔷 Object-Oriented Programming
- Classes and objects
- Constructors (default and parameterised)
- Inheritance and method overriding
- Encapsulation with getters and setters
- Interfaces and abstract classes
- Polymorphism

### 🔷 Core Java Concepts
- Data types, variables, and type casting
- Conditional statements (`if`, `else`, `switch`)
- Loops (`for`, `while`, `do-while`)
- Arrays and `ArrayList`
- String methods and manipulation
- Exception handling with `try-catch-finally`

### 🔷 Data Structures (Manual Implementations)
- Basic array-based structures
- Practice with collections

---

## 🚀 How to Run Any Program

### Option 1 — Terminal
```bash
git clone https://github.com/Talha8899/Java-practice.git
cd Java-practice
javac FileName.java
java FileName
```

### Option 2 — IDE
Open the project in **IntelliJ IDEA** or **VS Code** and run directly with the play button.

**Requirements:** Java JDK 8 or above
